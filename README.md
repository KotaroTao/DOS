# 百の迷宮と 魂の王 — Hundred Labyrinths: Rise of the Soul King

ウィザードリィ風の**カードめくり探索RPG**。HTML + ES Modules + Canvas だけで動く、ビルド不要の単一Webゲームです。モンスター・装備のイラストはすべて外部素材なしで、Canvas にドット絵をプログラム描画しています。

**▶ 今すぐ遊ぶ: https://kotarotao.github.io/DOS/**

スマホのブラウザで開いて「ホーム画面に追加」すれば、アプリのように起動・オフラインでも遊べます（PWA対応）。

## どんなゲーム？

死者の魂を喰らう百の迷宮。生身の人間は還れないその深淵へ、人型の器 **「人業（ドール）」** に死者の魂を宿して挑む——あなたは魂を繰る者、〈魂繰り〉。

- **百の迷宮** — 番号が進むほど深く険しい100のダンジョン。10迷宮ごとの層域 (ランク帯) で敵も戦利品も一段強くなり、**迷宮ごとに異なる固有の主 (ボス100体)** が最深部で待つ
- **カードめくり探索**（モンスターメーカー風）— 盤面のマスはすべて裏向きカード。めくって道を作り、モンスター・宝箱・罠・死体・泉・毒の床に出会いながら階段を探す
- **魂と人業の育成** — 死体から回収した魂を5部位（頭・両手・胴・足）に宿して人業を組む。同職3部位で職業が発現、3+2の組み合わせで**30種の混成職**が目覚める。魂のレベル・品質 (優秀/偉大/伝説) が職業ランクとパッシブを決め、宿す部位で魂の成長傾向が変わる（頭=精神・右手=攻撃・胴=生命など）
- **六大ステータス** — ATK/VIT/AGI/INT/PIE/LUK。装備8部位 + 属性攻撃/属性防御（火→風→土→水→火、光↔闇）の相性が戦闘を左右する
- **一点物のアイテムカタログ** — 来歴つきの固有装備・戦利品が400種以上。隠しレベル (1-200) により、深い迷宮でしか出ない品がある
- **街 (拠点)** — 人業の館（編成/魂封じ/強化/合成/分解）、酒場（クエスト・日替わり依頼・噂話）、商店（ボルタック式在庫）、宿屋、王宮（勅命=メインストーリー全100章、図鑑、勲章）、赤い魂の祠
- **ウィザードリィ風の緊張感** — 常時オートセーブでやり直し不可。全滅すれば戦利品を失い、砕けた人業は救出を待つ。呪いの装備、宝箱の罠（開封者を選ぶ）、ミミック、黒い泉・黒い宝箱の賭け
- **チップチューンBGMと効果音** — Web Audio で全曲をコード生成（施設/探索/戦闘/ボスで切替）

## 遊び方（ローカル）

ES Modules を使っているため、簡易サーバー経由で開きます（ビルド不要）。

```bash
python3 -m http.server 8000
# もしくは npx serve .
```

ブラウザで `http://localhost:8000/` を開いてください。

### 操作

| 操作 | 方法 |
|------|------|
| 移動（カードをめくる） | スワイプ / 隣のタイルをタップ / 矢印キー・WASD |
| 離れたマスへ自動移動 | めくり済みのマスをタップ（最短経路を自動で歩く） |
| 戦闘 | 敵をタップで攻撃 / 画面下のコマンド（呪文・防御・逃走・オート） |
| ステータス・装備 | 👤ボタン / パーティカードをタップ |
| ミュート切替 | 🔊ボタン / M キー |

## ファイル構成

```
index.html            画面レイアウト + 起動エラーの自己修復
styles.css            スタイル
manifest.webmanifest  PWA マニフェスト
sw.js                 Service Worker (バージョン一括キャッシュ)
src/
  game.js        メインループ / 入力 / 街・盤面・戦闘画面の統合 / セーブ
  board.js       辺に壁を持つ迷路生成 (行き止まりイベント・毒の床)
  combat.js      ターン制戦闘ロジック (Battle / SPELLS)
  sprites.js     ドット絵 (ヒーロー/アイコン) 定義と描画
  items.js       基本装備・装備ロジック (8部位)
  souls.js       魂と人業のデータモデル / 職業・混成職・ランクパッシブ
  audio.js       Web Audio による効果音とBGM生成
  story.js       メインストーリー (勅命100章)
  subquests.js   酒場のサブクエスト (迷宮×階ごとに決定的生成)
  opening.js     オープニング演出
  dungeons/      迷宮生成器 (100迷宮) + ランク別ベスティアリ + 固有ボス100体
  catalog/       一点物アイテムカタログ (武器/防具/小手/装飾/戦利品/伝説)
```

開発の詳しい約束事（ID の append-only 運用、Service Worker のバージョン更新など）は `CLAUDE.md` を参照してください。
