# 百の迷宮と 魂の王 - Hundred Labyrinths: Rise of the Soul King

ウィザードリィ風の擬似3DダンジョンRPG。HTML + JavaScript + Canvas だけで動く、ビルド不要の単一Webゲームです。モンスターのイラストは外部素材を使わず、Canvas にドット絵をプログラム描画しています。

**▶ 今すぐ遊ぶ: https://kotarotao.github.io/DOS/**

スマホのブラウザで開いて「ホーム画面に追加」すれば、アプリのように起動・オフラインでも遊べます（PWA対応）。

## 特徴

- **カードめくり探索**（モンスターメーカー風）— 盤面のすべてのマスが裏向きカード。隣のカードをめくると中身（道・壁・イベント）が判明する
- **辺に壁を持つ迷路**（縦6×横8）— 壁はマスを潰さず、隣との境界（辺）に表示される。分かれ道・行き止まりが多く、ルート次第ですべてのマスへ到達できる
- **行き止まりイベント**— イベント（モンスター・宝箱・罠・癒しの泉）は行き止まりにのみ50%で出現
- **各階に階段1つ**— スタートから10歩より遠い場所に隠れている
- **効果音とBGM**— Web Audio によるチップチューンを外部素材なしで生成（フィールド/戦闘でBGMが切り替わる、🔊でミュート）
- **タイルクリック / 矢印キー / ボタン**のどれでも移動可能
- **ドラクエ風の演出戦闘**— 素早さ順に1手ずつ進み、攻撃には「間」と効果音、敵の攻撃アクション（踏み込み）、斬撃・魔法・被弾エフェクト、ダメージ数値、画面フラッシュ
- **生死を分ける選択**— 階段は「降りる／まだ探索する」、宝箱は「開ける／開けない」を選択。宝箱には罠やミミックのリスクがあり、深い階ほど危険。**敵に勝利すると必ず宝箱が出現**
- **種族・属性**（ウィザードリィ風）— 各キャラに属性(善/中立/悪)・種族(人間/ドワーフ/エルフ等)。悪の装備は善のキャラには装備できない等の制限
- **ターン制パーティ戦闘**（最大6人／素早さ順の個別ターン制／攻撃・魔法・被弾エフェクトとドラクエ風の間）
- **個別ステータス＆装備画面**— 装備は頭・武器・盾・防具・小手・足・装飾2の**8部位**、各キャラ**所持品12個**まで。武器防具はドット絵イラストと説明文つき
- **ウィザードリィ風要素**— 職業制限・両手武器・呪いの装備・AC(アーマークラス)・状態異常「毒」・消耗品（薬草/毒消し/マナの雫）
- **宝箱の入手演出**— 装備やアイテムを得るとイラスト込みの獲得画面で表示
- **コードで描くドット絵モンスター**（スライム / バット / コボルド / スケルトン / オーク / レイス / ドラゴン）
- **呪文システム**（ハリト、マハリト、ディオス、ディアル、カティノ）
- **経験値とレベルアップ**、階段を降りて深部のドラゴンを討つと制覇
- **スマホ縦型UIに最適化**— 画面いっぱいに盤面を表示し、ページスクロールなし。下部に親指で押しやすい十字キー、戦闘は大きなコマンドボタン。セーフエリア（ノッチ）対応
- **PWA 対応**（manifest + Service Worker）。ブラウザから「ホーム画面に追加」でアプリのように起動・オフラインでも遊べます

## 遊び方

ローカルで動かすには、ES Modules を使っているため簡易サーバー経由で開きます。

```bash
# どれか1つ
python3 -m http.server 8000
# もしくは
npx serve .
```

ブラウザで `http://localhost:8000/` を開いてください。

### 操作

| 操作 | 方法 |
|------|------|
| 1マス移動（カードをめくる） | 隣のタイルをタップ / 矢印キー・WASD / ▲▼◀▶ボタン |
| 離れたマスへ自動移動 | めくり済みのマスならタップで最短経路を自動移動（未公開カードは勝手にめくらない） |
| ミュート切替 | 🔊ボタン / M キー |
| 戦闘 | 画面下のコマンドボタン |

隣のマスへ進むと裏向きのカードがめくられます。壁はマスの**辺**にあり、開いている辺の方向にだけ進めます（行き止まり・分かれ道ができます）。すべてのマスはいずれかのルートで到達可能です。行き止まりにはモンスター・宝箱・罠・癒しの泉のいずれかが潜むことがあります。各階に1つある階段（スタートから10歩より遠くに隠れる）を見つけると次の階へ。B3F の階段にはドラゴンが待ち構えており、倒せばクリアです。戦闘では素早さ順に手番が回り、各メンバーの行動（攻撃 / 呪文 / 防御 / 逃走）を選びます。逃走すると元のマスに戻ります。

## ファイル構成

```
index.html            画面レイアウト
styles.css            スタイル
manifest.webmanifest  PWA マニフェスト
sw.js                 Service Worker(オフライン)
icon.svg              アプリアイコン
src/
  game.js       メインループ / 入力 / 盤面⇄戦闘の統合
  board.js      辺に壁を持つ迷路生成(成長木法/行き止まりイベント/階段配置)
  sprites.js    ドット絵(モンスター/ヒーロー/アイコン)定義と描画
  combat.js     パーティ・呪文・ターン制戦闘
  items.js      武器/防具/装飾/消耗品カタログ・装備ロジック(8部位)
  audio.js      Web Audio による効果音とBGM生成
```
